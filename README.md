# BEFORE YOU START

1. make sure you have `emcc` SDK on your machine https://kripken.github.io/emscripten-site/docs/getting_started/index.html

2. `$ nvm use` from the root of the project



## EXPERIMENT 1

to start:

`$ cd ./exp1`

`$ sh ./build.sh`

`$ npm install -g http-server`

`$ http-server`


## EXPERIMENT 2

to start:

`$ cd ./exp2`

`$ npm install`

`$ npm start`
